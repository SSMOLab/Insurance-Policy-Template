## 安裝執行專案

### 1. 確認安裝 [Node.js](https://nodejs.org/zh-tw/download/) 環境

### 2. 複製專案到本機電腦

```
$ git clone https://github.com/SSMOLab/Insurance-Policy-Template
```

### 3. 執行後端服務

動態資料與圖檔皆存放於 `app/publc/` 內，需先切換目錄至 `app/` 後以 `npm run start` 執行後端服務才可取得。

```
cd app
npm install
npm run start
```

### 4. 執行前端

```
npm install
npm run serve
```
